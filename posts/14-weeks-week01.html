Week 1: project setup & governing equations

Part of the 14 weeks to a Shallow Water solver series.

Before we start

  • Install: Python 3.11+, NumPy, Matplotlib; optional: Jupyter, git, VS Code (or your preferred IDE).
  • Prereqs: basics of shallow water equations, calculus and linear algebra, finite differences, and comfort with Python scripting.
  • Goal: a clean, minimal solver that hydraulic engineers can read, modify, and extend (sediment transport, source terms, parallelization, GPU, etc.). Think of it as a bridge between “black-box” models and the equations underneath.

What we will do in Week 1

This week is about setting the foundations: creating a simple, reproducible environment and writing down the shallow water equations we want to solve in 2D. We will:

  • Set up a minimal project structure (folders for code, notebooks, and figures).
  • Decide on units, notation, and conventions for depth, velocity, and discharge.
  • Write the conservative form of the shallow water equations and identify all terms we plan to include (and which we will postpone).

The actual code and notebooks for Week 1 will be pushed progressively to the project repo as the project advances.
