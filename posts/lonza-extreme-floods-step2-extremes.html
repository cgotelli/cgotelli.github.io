Lonza floods – Step 2: extreme value analysis of flows

Project hub: Estimating extreme floods in an alpine catchment

Choosing an extreme value framework

To estimate a 100‑year flood from observed flows, I apply tools from extreme value theory. There are two classical approaches:

  • Annual maxima series (AMS): select one maximum per year, then fit a Generalized Extreme Value (GEV) distribution (or, in simpler cases, a Gumbel distribution).
  • Peaks‑over‑threshold (POT): select all independent peaks above a high threshold, and fit a Generalized Pareto distribution (GPD) to the excesses.

For a small alpine basin like the Lonza, the POT approach often extracts more information from a relatively short record, provided we carefully decluster events.

Selecting peaks and fitting the model

In practice, I:

  • Choose a threshold so that we retain enough peaks (for example, 5–10 per year) without violating independence.
  • Apply a minimum time separation between peaks to avoid counting the same event multiple times.
  • Fit the GEV or GPD model using maximum likelihood and inspect diagnostic plots (QQ‑plots, return level plots, stability of estimates versus threshold).

This process leads to an estimate of the 100‑year return level \(Q_{100}\), together with an uncertainty band (e.g. a 95% confidence interval).

First estimate of \(Q_{100}\)

The outcome of this step is a first, data‑driven estimate of the design peak flow, rooted directly in the gauged behaviour of the river. In the next step, I will confront this flow‑based estimate with a rainfall‑driven perspective using a conceptual rainfall–runoff model, and check their consistency.
